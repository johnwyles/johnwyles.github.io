Setting Up Kubernetes on a Raspian Cluster

Mar 25, 2019
3 min read

NOTE: This guide was based on a gist by GitHub user alexellis for setting up Kubernetes on Raspbian as well as a gist getting started with Raspian for each node which I have made some minor additions to in this post.

Assumptions

  • You have more than 1 Raspberry Pi with at least an 8GB SD card for each
  • You want all of your Raspberry Pi’s to have a static IP
  • You desire all of your networking and orchestration over WIFI rather than Ethernet
  • For troubleshooting that you have access to your routers administrative panel to see attached devices and their IP address
  • For further troubleshooting you may require Ethernet access for diagnosis and further setup

Steps

  1. Download the latest Raspbian Lite image from here (as of writing this was: 2018-11-13 Raspbian Stretch Lite ): Raspbian Images

  2. If you are on a Mac use a software like balena Etcher to flash the image to only one of the Raspberry Pi SD cards that will be in your cluster. This machine will allow us to image all of the subsequent Raspberry Pi machines in this cluster.

    Setup

  3. Copy and past the following contents into a file named make-rpi.sh on your machine:

#!/bin/bash

export DEV=sda
export IMAGE=2018-11-13-raspbian-stretch-lite.img

if [[ $EUID -ne 0 ]]; then
   echo "This script must be run as root"
   exit 1
fi

if [ -z $1 ]; then
  echo "Usage: ./make-rpi.sh <hostname> <ip-suffix>"
  echo "       ./make-rpi.sh node-1 101"
  echo "       ./make-rpi.sh node-2 102"
  exit 1
fi

if [ -z "$SKIP_FLASH" ];
then
  echo "Writing Raspbian Lite image to SD card"
  time dd if=$IMAGE of=/dev/$DEV bs=1M
fi

sync

echo "Mounting SD card from /dev/$DEV"

sudo mkdir -p /mnt/rpi/boot
sudo mkdir -p /mnt/rpi/root

sudo umount /dev/sda2

mount /dev/${DEV}1 /mnt/rpi/boot
mount /dev/${DEV}2 /mnt/rpi/root

# Add our SSH key
mkdir -p /mnt/rpi/root/home/pi/.ssh/
cat template-authorized_keys > /mnt/rpi/root/home/pi/.ssh/authorized_keys

# Enable ssh
touch /mnt/rpi/boot/ssh

# Disable password login
sed -ie s/#PasswordAuthentication\ yes/PasswordAuthentication\ no/g /mnt/rpi/root/etc/ssh/sshd_config

echo "Setting hostname: $1"
sed -ie s/raspberrypi/$1/g /mnt/rpi/root/etc/hostname
sed -ie s/raspberrypi/$1/g /mnt/rpi/root/etc/hosts

# Reduce GPU memory to minimum
echo "gpu_mem=16" >> /mnt/rpi/boot/config.txt

# Set WIFI and static IP
cp /mnt/rpi/root/etc/dhcpcd.conf /mnt/rpi/root/etc/dhcpcd.conf.orig
sudo cp template-wpa_supplicant.conf /mnt/rpi/root/etc/wpa_supplicant/wpa_supplicant.conf
sed s/100/$2/g template-dhcpcd.conf > /mnt/rpi/root/etc/dhcpcd.conf

echo "Unmounting SD Card"

umount /mnt/rpi/boot
umount /mnt/rpi/root

sync
